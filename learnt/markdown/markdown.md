# Markdown Syntax

## Headings

```markdown
# heading level 1
## heading level 2
### heading level 3
...
###### heading level 6
```

> # heading level 1
> ## heading level 2
> ### heading level 3
> ...
>
> ###### heading level 6

## Horizontal Rules

```markdown
up
---
down
```

> up
>
> ---
>
> down

## Paragraphs and Line Breaks

Paragraphs: use **blank line** to separate

Line break: `<br>`

```markdown
para1

para2,line1<br>
line2
```

> para1
>
> para2,line1<br>
> line2

## Emphasis

```markdown
**bold**
*italic*
***bold and italic***
```

> **bold**
> *italic*
> ***bold and italic***

## Blockquotes

```markdown
> blockquotes paragraph1
>
> blockquotes paragraph2
>
>> nested blockquotes
```

>  > blockquotes paragraph1
>  >
>  > blockquotes paragraph2
>  >
>  > > nested blockquotes

## Lists

```
1. order list
1. order list
2. order list

- unorder list
- unorder list
```

> 1. order list
> 1. order list
> 2. order list
>
> - unorder list
> - unorder list

## Code

````markdown
`print('hello world')`

```python
print('hello world')
```
````

> `print('hello world')`
>
> ```python
> print('hello world')
> ```

## Hyperlink

```markdown
[Google.com](https://www.google.com/)
[Google.com](https://www.google.com/ "search engine")
<https://www.google.com/>
<yuanpinzhou@gmail.com>
```

> [Google.com](https://www.google.com/)
> [Google.com](https://www.google.com/ "search engine")
>
> <https://www.google.com/>
> <yuanpinzhou@gmail.com>
